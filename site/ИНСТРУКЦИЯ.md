# 🚀 HP Тюнинг — Инструкция по деплою на Netlify

## Как опубликовать сайт

### Способ 1 — Drag & Drop (самый быстрый)

1. Перейти на https://app.netlify.com/
2. Войти в аккаунт (или зарегистрироваться)
3. На главной странице найти блок **"Deploy manually"** → **"drag and drop your site folder here"**
4. Перетащить **всю папку `site/`** прямо в браузер
5. Netlify сам опубликует сайт и выдаст URL вида `https://xxxxxx.netlify.app`

> ✅ Готово! Сайт уже живёт в интернете.

---

### Способ 2 — через GitHub (для обновлений)

1. Создать репозиторий на GitHub
2. Загрузить туда содержимое папки `site/`
3. На Netlify: **Add new site → Import an existing project → GitHub**
4. Выбрать репозиторий, **publish directory** = `.` (точка — публикуем корень)
5. Нажать **Deploy**

> При каждом `git push` сайт обновляется автоматически.

## После деплоя — настроить домен

1. В Netlify: **Site settings → Domain management → Add custom domain**
2. Ввести `hp-tuning.ru`
3. Прописать DNS у регистратора:
   - Тип: `CNAME`, имя: `www`, значение: `xxxxxx.netlify.app`
   - Или `A`-запись на IP Netlify (покажет в интерфейсе)
4. SSL выдаётся автоматически через Let's Encrypt — бесплатно
