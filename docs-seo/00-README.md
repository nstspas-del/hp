## 🎯 Как пользоваться пакетом

Этот архив содержит **всё**, что нужно агенту для создания и оптимизации SEO-контента на поддоменах HP Тюнинг. Работаем **по брендам**, начиная с BMW (самый маржинальный).

### Порядок работы с одним брендом (пример BMW):

1. Открыть `semcore/BMW-semcore.md` → взять ВЧ/СЧ/НЧ запросы и заголовки.
2. Открыть `01-BMW.md` → следовать промту, писать текст 3500-4500 символов.
3. Получить от Димы (через `briefs/DIMA-INTERVIEW-QUESTIONS.md`):
   - 5–7 реальных фото работ по BMW
   - 3–5 свежих отзывов с именами
   - Прайс 3–5 услуг
   - 15-минутное голосовое интервью
4. Вставить реальные данные в текст → отправить на вычитку Димы/Насти.
5. Передать бэкенд-агенту → публикация на `bmw.hptuning.ru`.
6. Мониторить Яндекс.Вебмастер 2-3 недели, корректировать.

## 🎨 Что делает каждый тип файла

### `00-TEMPLATE-BRAND.md`
Универсальный шаблон. Если в будущем добавится 9-й бренд (Jaguar, Bentley, Mini), копируем этот файл и заполняем.

### `0X-BRAND.md` (брендовые промты)
Содержат:
- Описание бренда и целевой аудитории.
- Какие услуги продавать.
- Тон голоса (premium / народный).
- Структуру H1-H2-H3.
- Требования к тексту: длина, LSI, запреты.
- Примеры кейсов.
- Requirement: Schema.org разметка.

### `semcore/*.md` (семантические ядра)
- ВЧ (>1000 показов) — для H1/H2.
- СЧ (200-1000) — для H2/H3 и разбиения на секции.
- НЧ (<200) — для FAQ и длинного хвоста в тексте.
- Гео-запросы по районам СПб.
- ТОП-10 конкурентов в SERP.
- Ценовой диапазон по рынку.
- Голосовые запросы для FAQPage.
- LSI-слова для естественного вхождения.
- YATI-рекомендации Яндекса.

### `briefs/DIMA-INTERVIEW-QUESTIONS.md`
Готовый опросник на 15-20 минут. Дима отвечает голосом/текстом, мы расшифровываем и вставляем цитаты в тексты (это даёт E-E-A-T и выделяет от конкурентов).

### `MASTER-PROMPT-AUTODEALER.md`
Технический промт бэкенд-агенту. Блокирует запуск Я.Директа, если не выполнен.

## 🛡️ Важные запреты для агента

❌ **Не писать SEO-полотно** — каждый абзац должен содержать смысл для реального клиента.  
❌ **Не придумывать цены** — только реальные от Димы.  
❌ **Не придумывать отзывы** — только скопированные с Я.Карт или 2ГИС.  
❌ **Не использовать стоковые фото** — Яндекс распознаёт и понижает.  
❌ **Не переспамливать ключевиками** — YATI за это понижает в выдаче.  
❌ **Не использовать стоп-фразы Яндекса:** «гарантируем 100% результат», «самая низкая цена в СПб», «лучшие в городе» (без подтверждения).  
❌ **Не копировать текст с сайтов конкурентов** — уникальность ≥ 90% по text.ru.
