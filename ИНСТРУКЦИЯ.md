# Инструкция по установке SEO-исправлений для HP Тюнинг

## ⚡ Критические исправления (сделать ПЕРВЫМ делом)

### 1. Вставить <head> в index.html
Откройте ваш `site/index.html`. Сразу после `<html lang="ru" ...>` вставьте 
содержимое файла `head-snippet.html`.

**ВАЖНО:** 
- Замените `ВСТАВЬТЕ_КОД_ВЕРИФИКАЦИИ` на код из Яндекс.Вебмастера
- Раскомментируйте строку с подключением CSS и укажите правильный путь
- Если домен не hptuning.ru — замените все URL на ваш домен

### 2. Загрузить robots.txt
Положите `robots.txt` в КОРЕНЬ сайта (рядом с папкой `site/`, не внутри неё).

### 3. Загрузить sitemap.xml
Положите `sitemap.xml` в КОРЕНЬ сайта.

### 4. Загрузить _headers (необязательно)
Положите `_headers` в КОРЕНЬ сайта для оптимизации кеширования на Netlify.

## ⚠️ Удалить спам-абзац
В конце index.html найдите и УДАЛИТЕ этот блок:
```
<p>Автосервис в Порошкино СПб. Чип-тюнинг в Санкт-Петербурге...</p>
```
Это скрытый текст с ключевыми словами — Яндекс может наложить фильтр «Минусинск».

## 📋 Следующие шаги
1. Зарегистрируйтесь в Яндекс.Вебмастере → добавьте сайт → получите код верификации
2. Зарегистрируйтесь в Яндекс.Метрике → создайте счётчик → вставьте код в <head>
3. Создайте страницу contacts.html с адресом, реквизитами, Яндекс.Картой
4. Добавьте alt-тексты ко всем изображениям
5. Добавьте loading="lazy" ко всем изображениям кроме hero
